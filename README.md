# AI Physics Education Platform

This monorepo contains the scaffolding for a scalable, AWS-deployable AI-powered physics education platform.

## Structure
- frontend: React app for user interface
- backend: FastAPI (Python) for API and business logic
- ai-agents: Microservices for AI/agentic professors
- infra: Infrastructure as Code (AWS CDK/Terraform)
- docs: Documentation
- .github: Project automation and instructions

## Getting Started
Each folder contains its own README and setup instructions. See /docs for architecture and onboarding.
